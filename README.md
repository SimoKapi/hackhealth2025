# Hackhealth 2025 submission (MSP Hub)

A Ventricular Assist Device (VAD) monitor and data telemeter

## Technical Features

* **Optical Character Recognition:** Used Tesseract.js for OCR handling. The input image had to be straightened, contrast boosted, and optimized using OpenCV.
* **Raspberry Pi portable device:** Made for a Raspberry Pi with a Raspberry Pi Camera Module. Designed to be clipped onto the handle of VAD.

---

## Setup & Usage

### 1. Configuration
To install dependencies:

```bash
bun install
```

### 2. Execution
To run:

```bash
bun run index.ts
```
